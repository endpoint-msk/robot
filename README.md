# endpoint-robot

Телеграм-бот для хакерспейса на TypeScript + [mtcute](https://mtcute.dev).
Ведёт ежемесячные сборы донатов и отслеживает, кто из резидентов сейчас в спейсе.

## Что умеет

### Резиденты (присутствие в спейсе)

Резиденты — админы подключённых чатов.

- `/start` в личке — меню: «Отметиться с ником» / «Отметиться без ника».
- При отметке во все чаты, где резидент — админ, постится список присутствующих
  (`Внутри [6], отметились [N]: …`).
- Каждые 3 часа бот пингует в личку «Ты ещё в спейсе?». Кнопка «Я внутри»
  продлевает отметку. Если за 15 минут кнопка не нажата — отметка снимается,
  список в чатах обновляется.
- Если в чате нет сообщений ≥ 5 часов и есть отмеченные — бот сам пушит
  актуальный список.

### Сборы

- Каждый календарный месяц — отдельный сбор.
- `/goals` — показать текущий сбор. К сообщению цепляется кнопка **«Обновить»**, а если донатящих больше одной страницы — стрелки `◀️ ▶️`.
- `/donate <сумма> <ник>` — добавить донат в текущий сбор.
- `/remove <номер>` — удалить все донаты участника, занимающего это место в лидерборде.
- `/remove <ник> [сумма]` — удалить один донат по нику (и опционально сумме).
- `/setgoal <сумма>` — задать цель текущего сбора (0 — снять цель).
- `/settitle <тема>` — изменить тему сбора (по умолчанию «аренду»).
- `/help` — список команд.
- При нажатии **«Обновить»** или при `/donate` сообщение перерисовывается на месте; «Обновить» всегда возвращает на первую страницу.
- Список донатящих — лидерборд по сумме (с учётом нескольких донатов одного ника), топ-3 помечены 🥇🥈🥉.
- При перевыполнении цели в сообщении появляется пометка «✅ Сбор закрыт».
- При наступлении нового месяца последнее «живое» сообщение в каждом чате
  автоматически переключится на новый сбор (проверка раз в минуту).
- Бот работает только в чатах из allowlist (`ALLOWED_CHATS`); в остальных молча игнорирует команды.
- Команды и кнопка доступны только админам/владельцу той группы, где они вызваны
  (через `getChatMember`).

## Запуск

```bash
cp .env.example .env   # заполнить API_ID/API_HASH/BOT_TOKEN/ALLOWED_CHATS
npm install
npm start
```

## Структура

- `src/index.ts` — точка входа: стейт, клиент, диспетчер, шедулеры.
- `src/handlers.ts` — команды сборов и callback пагинации/обновления.
- `src/fundraiser.ts` — модель сбора, лидерборд, прогресс-бар, форматирование.
- `src/presence.ts` — присутствие резидентов: меню, пинги, тишина в чатах.
- `src/storage.ts` — JSON-хранилище с атомарной записью.
- `src/scheduler.ts` — фолбэк-таймер, перерисовывающий сообщение при смене месяца.
- `src/types.ts` — типы стейта.

Стейт лежит в файле, заданном `DATA_FILE` (по умолчанию `./data.json`),
сессия mtcute — в `bot.session` (создаётся в рабочей директории).
